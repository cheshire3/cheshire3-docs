Cheshire3 Objects: Parser
Description

Normally a simple wrapper around an XML parser, these objects can be viewed as Record Factories. They take a Document containing some XML and produce the equivalent Record.

Implementations
  • parser.SaxParser
    Default SAX2 expat based parser
  • parser.MinidomParser
    Python standard library's Minidom parser
  • parser.FtParser
    Fourthought's nonvalidating DOM parser
  • parser.LibxmlSaxParser
    Libxml2 Sax parser
  • parser.DTDParser
    Libxml2 DTD validating parser
  • parser.RelaxNGParser
    Libxml2 RelaxNG validating parser
  • parser.LibxmlParser
    Libxml2 DOM parser
  • parser.XmlRecordStoreParser
    Parser to create SAX records from the default record store implementation
